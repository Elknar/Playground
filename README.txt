Retrieve.py
Simple script to download a Reddit header image, using provided URL.

Requires Python 2.x with urllib2 and sys modules (preferable Python 2.7)
It is possible to download the latest version for supported platforms at: http://www.python.org/download/

Usage:
Type the following in the command line:
python path_to_Retrieve.py "URL"

As an alternative, retrieve_manual.py can be launched without arguments and you may provide the URL upon request.
